# SSES-001 : GPIO Standard

Version: 1.0

## Purpose

यह दस्तावेज़ Sarva Siddhi के सभी SH (Sarva Hardware) मॉड्यूल के लिए GPIO उपयोग का मानक निर्धारित करता है।

---

## Communication

GPIO21 : I2C SDA

GPIO22 : I2C SCL

GPIO16 : UART RX2

GPIO17 : UART TX2

GPIO18 : SPI SCK

GPIO19 : SPI MISO

GPIO23 : SPI MOSI

GPIO5 : SPI CS

---

## Outputs

GPIO25 : Relay 1

GPIO26 : Relay 2

GPIO27 : Relay 3

GPIO13 : PWM 1

GPIO14 : PWM 2

---

## Inputs

GPIO32 : Digital Input 1

GPIO33 : Digital Input 2

GPIO34 : Digital Input 3

GPIO35 : Digital Input 4

GPIO36 : ADC 1

GPIO39 : ADC 2

---

## Status

GPIO2 : Status LED
